# C# and .NET Framework Projects

This repository contains my C# and .NET Framework course projects and assignments.

These projects helped me practice object-oriented programming, ASP.NET MVC, Entity Framework, Code First development, and working with GitHub.

## Projects

- Car Insurance
- Operators Assignment
- Polymorphism Assignment
- Code First Student Database

## Car Insurance

This project is an ASP.NET MVC web application that mimics a car insurance quote website. It takes the user's input, calculates a monthly quote based on business logic, and stores the information using Entity Framework.

The project also includes an Admin page that allows staff to view all quotes that have been issued, including the user's first name, last name, email address, and quote amount.

## Operators Assignment

This project demonstrates the use of C# operators and operator overloading. It helped me practice comparing objects and using custom logic in classes.

## Polymorphism Assignment

This project demonstrates polymorphism in C#. It includes an Employee class and shows how interfaces and inheritance can be used in object-oriented programming.

## Code First Student Database

This project is a basic Entity Framework Code First console application. It creates a Student database and adds one student using C# classes and a database context.
